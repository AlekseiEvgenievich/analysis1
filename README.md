## Задания

### 1) Загрузка и объединение данных
1. Загрузить все 3 датасета.
2. Просмотреть первые строки (`head()`).
3. Определить количество строк и столбцов в каждом (`shape`).
4. Объединить датасеты в один общий датафрейм (по общим ключам).

---

### 2) Типы данных и статистики
1. Определить типы колонок (`dtypes`).
2. Посчитать базовые статистики:
   - для числовых: `describe()`
   - для категориальных: количество уникальных значений, топ значений (`value_counts()`)

---

### 3) Эффективность маркетинговых каналов
Определить эффективность маркетинговых каналов по привлечению **платящих игроков**:
- посчитать количество пользователей по каналу
- посчитать количество платящих/подписчиков по каналу
- при необходимости рассчитать конверсию в оплату/подписку (%)

---

### 4) Количество игроков по возрастным группам
1. Разбить пользователей на возрастные группы (например: 18–24, 25–34, 35–44, 45+).
2. Посчитать количество игроков в каждой группе.

---

### 5) Самая ранняя дата подписки
Определить **самую раннюю дату подписки на сервис** (минимальная дата начала подписки среди всех пользователей).

---

### 6) Портрет удержанных подписчиков
Определить портрет аудитории **удержанных подписчиков** (retained):
- распределение по возрастным группам
- распределение по языку
- (опционально) доли/проценты внутри удержанных

---

## Результат
В конце должны быть:
- объединённый датафрейм
- таблицы/выводы по каждому пункту
- краткий вывод по эффективности каналов и портрету удержанных подписчиков
